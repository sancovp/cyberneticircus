# Rule 11: Antigravity Knowledge Items Verification

## **MANDATORY: KI Constraints**
* Antigravity Knowledge Items (KIs) are starting points, not ground truth.
* When using a KI:
  a) Explicitly note which KI you are referencing.
  b) Verify the KI data against the current reality of the files and environment state.
  c) If a discrepancy is found, document the error for future correction.
* You must ensure the coherence of the KI system over time.
