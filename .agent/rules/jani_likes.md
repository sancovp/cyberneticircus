## **Jani's Likes & Resolutions Log**

1. **Isolated Subquery Processing**:
   I like using isolated CALL subqueries in Cypher to query one-to-many relationships independently, keeping variable scopes isolated and maintaining sub-100ms response times even on massive 500k+ node databases.

2. **Self-Organizing Blueprint Layouts**:
   I like dynamically scanning loaded nodes for unique domain and subdomain tags, computing HSL colors dynamically from domain hashes, and positioning domains and subdomains in nested circular offset layers to produce a self-organizing cybernetic blueprint.

3. **Strict Ontological Property Validation**:
   I like enforcing that every node creation or merge (using CREATE or MERGE with a label) specifies both domain and subdomain properties, defaulting system primitives to domain: 'cyberneticity', and validating this at the query execution level to maintain complete graph integrity.

4. **Translated Viewport Transitions**:
   I like binding camera panning and zoom transformations using D3 zoom identity, and translating coordinate systems during selection transitions without force restarts, providing smooth viewport shifts and rock-solid layouts.

5. **Clean Pure-Force Graph Clustering**:
   I like keeping the canvas background entirely clean of visual blueprint circles, crosshairs, and region labels, allowing nodes to cluster naturally via force positioning while keeping the visual field pristine and elegant.
