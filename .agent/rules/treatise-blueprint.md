# Rule: Treatise Blueprint and Victory Spires

## **Purpose**
Defines the final completion scope for Jani's development sprint, directing the creation of the holographic treatise, click-to-copy interactive tutorial course, and full-window visualizer.

## **MANDATORY: Spires of Completion**
1. **The Treatise of Metashifting**: Document the exact methodologies to seed a Jani shell, configure its `.agent` directory, and structure the system prompt hierarchies for autopoietic recursion.
2. **Interactive Course (The Book UI)**: Implement a browser page rendering the chapters of the book "Weights of Time". Every step in the tutorial must be a click-to-copy block that can be pasted into the terminal, with a visual checklist to track user progress.
3. **Observer Canvas**: Set the D3 graph visualizer as a full-window background layer behind the text. Ensure node ticks and path traversals animate in the background while the user reads and copies prompts from the foreground layers.

## **Triggers**
* Building web pages, books, or documentation UIs.
* Defining final simulation runs or agent runner delegations.
