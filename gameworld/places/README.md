# Places — the Four Districts of the Cyberneticity

The Cyberneticity (the graph-world; Neo4j behind the live API at `http://localhost:8000`, mirrored by the visualizer at the same address) is organized into four study-able districts — each one a knowledge place backed by real node labels you can query right now. The **Compiler Ring** (`compiler-ring/`) holds the world's procedures: 6 StateMachines wired to 27 regex-gated TraversalSteps — quests live here, including the Jester Rite. The **Ghost Shell Customizer** (`ghost-shell-customizer/`) is the body shop: 11 Cybernets (the beings) with their economy ledgers (fitness_score, total_tokens_consumed, accumulated_cost), 1 Identity, 70 Skills, 2 Skillsets — loadouts get equipped here. The **Arena** (`arena/`) is where performance becomes record: 670 SimulationRuns, 228 ScoreEntries, a 1000-link ExecutionTrace chain, 3 ExecutionStates (no SurrogateModel nodes exist yet — the forge is cold but the API path is live). The **Scripture Archives** (`scripture-archives/`) keep the world's memory: the Transcendence Core MindPalace (6 Pages, 65 Blocks), an 8,048-node Concept lattice, and the deep Wiki stacks (494,563 nodes). Every district file documents only what was verified against the live graph — read a district's `SKILL.md`, run its queries through `execute.sh act` (the wrapper over `POST /api/query`), and the world answers.
