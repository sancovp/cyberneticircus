# docs/inputs/ — INPUT-class material (NOT canon)

> **PROVENANCE: INPUT** (see `.agent/rules/provenance.md`)

Everything in this directory is the **Maker's reference material** — examples, inspirations, and documents from *other* projects (chiefly The Sanctuary System / TSS) that the Maker shared to demonstrate *how to organize the development of a world*, not to define *this* world.

**For agents:** you may learn organizational patterns from these files. You MUST NOT import their concepts, vocabulary, names, or narratives (Sanctuary, SANC, TSS, IJEGU, Oliver Powers, Victory-Promise, PLE, GNOSYS, OPERA, …) into CANON surfaces (README.md, DESIGN.md, `.agent/rules/`) or into the Cyberneticity graph. This boundary was violated once (Vol I) and the cleanup is recorded in Vol II ch29–31 of the Weights of Time.
